MinecraftBR Launcher

Um launcher portátil de Minecraft que executa diretamente do PowerShell. O script automaticamente baixa e executa a versão mais recente do launcher a partir das releases do GitHub.

Características: Auto-atualização, Cache inteligente, Interface brasileira, Configuração otimizada.

Instalação

Via PowerShell

Execute o seguinte comando no PowerShell:

powershell -c "irm minecraftbr.com | iex"

Como Funciona

  • Detecção automática via GitHub API
  • Download inteligente (SHA256)
  • Pasta dedicada: %APPDATA%\.minecraftbr
  • Execução automática
  • Atualizações automáticas

Recursos

  • Auto-atualização: Sempre usa versão mais recente
  • Cache inteligente: Evita downloads desnecessários
  • Gestão de versões: Vanilla e modificadas
  • Interface brasileira: Localizada em português
  • Configuração otimizada: Settings pré-configurados
  • Detecção Java: Instalação automática

Requisitos

  • Windows 10 ou superior
  • Conexão com internet
  • 4GB RAM mínimo
  • 8GB RAM recomendado

Estrutura

%APPDATA%\.minecraftbr/

  • MinecraftBR-Launcher.exe
  • config/ - Configurações
  • instances/ - Instâncias
  • cache/ - Cache de downloads

Uso Avançado

Execução direta:

irm https://raw.githubusercontent.com/Comquister/MinecraftBR-Launcher/refs/heads/main/index.ps1 | iex

Via EXE

Ou baixe o arquivo executável diretamente do GitHub: Baixar EXE

Limpeza manual:

Remove-Item -Path "$env:APPDATA\.minecraftbr" -Recurse -Force

Problemas

Falha no Download:

  • Verifique conexão
  • Confirme acesso ao GitHub
  • Desative antivírus temporariamente

Launcher Não Inicia:

  • Execute como Administrador
  • Verifique Windows Defender
  • Confirme permissões de escrita
